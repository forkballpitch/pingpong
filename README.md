# pingpong


### GO

1. 구글이 개발한 가장 핫한 언어
1. Go를 사용중인 회사들
1. uber의 Go언어 사용사례, parse의 ruby에서 go로 전환사례(루비야 아프지마.. ㅠ)
1. 코딩이 재밌고 속도도 괜찮음
1. 정적언어지만 동적언어의 장점을 많이 가져옴
1. 빠른 컴파일 빠른 실행
1. 가비지컬랙션, 메모리 처리 안정성(nil 참조등), methods, interfaces, type assertions, reflection 지원
1. VM사용안함. 바로 실행가능한 바이너리 파일로 생성됨
1. 심플하고 간결한 문법
1. 클래스가 없고
1. 상속이 없고
1. 생성자가 없고
1. final이 없고
1. exception이 없고
1. annotation이 없고
1. generic이 없음
1. 쉽고 훌륭한 동시성 처리
1. code style을 강제하여 협업과 유지보수에 유리함
1. 단점으로 검색이 어렵고 (구글에서 go를 입력해보자) 패키지관리 시스템이 npm, gem처럼 안정되지 않음
